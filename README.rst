Installation (Coming Soon!)
----------------------------

mom6-tools can be installed from PyPI with pip:

.. code-block:: bash

    pip install mom6-tools


It is also available from `conda-forge` for conda installations:

.. code-block:: bash

    conda install -c conda-forge mom6-tools
